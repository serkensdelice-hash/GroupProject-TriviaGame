# 🎮 Trivia Game App (Group Project)

A mobile Trivia Game application built using SwiftUI that allows users to customize quiz settings and test their knowledge through an interactive quiz experience.

---


---

#  Brainstorming (Activity 1)

## Initial Ideas
1. Trivia Game App  
2. Fitness Tracker App  
3. Budget Manager App  
4. Language Learning App  
5. Event Planner App  
6. Study Flashcard App  

## Top 3 Ideas Evaluation

| Idea | Pros | Cons |
|------|------|------|
| Trivia Game | Interactive, API-based | Requires networking |
| Fitness Tracker | Useful, practical | Requires sensors |
| Budget App | Real-world use | Complex data handling |

## Final App Idea
👉 **Trivia Game App** was selected because it allows implementation of networking, UI design, and user interaction.

---

# 📱 App Overview (Activity 2)

The Trivia Game App allows users to customize quiz settings such as difficulty, number of questions, and type. The app fetches questions dynamically and provides an interactive quiz experience with a final score display.

---

# 📋 App Specification

## User Features
- Select number of questions  
- Choose difficulty level  
- Choose question type  
- Answer quiz questions  
- View final score  

---

## Screens & Navigation

1. **Options Screen**
   - Select quiz settings  
   - Start quiz  

2. **Quiz Screen**
   - Display questions  
   - Answer input  
   - Timer (if implemented)  

3. **Results Screen**
   - Final score  
   - Restart option
